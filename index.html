API Samples
Approval tool - Collection viewer
This sample demonstrates how to update metadata, for example to update a status, change a rating or enter comments. It incorporates dynamic thumbnail rendering and a full-screen preview lightbox that supports swipe gestures on mobile devices. Also demonstrates and how to search and display the contents of a collection.
Live feed
This sample demonstrates a live feed of incoming material. It auto-refreshes at a configurable interval. It also greys out thumbnails by date, giving a visual indication of the age of items from the feed.
Syndication site - DAM photo
This sample demonstrates how to filter a search with various facets. It uses an off-the-shelf lightbox to display previews, supporting both images and videos. Also demonstrates result paging and custom thumbnail buttons.
Upload portal - DAM tube
This sample demonstrates how to allow your site visitors to upload files directly into elvis. You can upload videos and images and browse and play uploaded material. Also demonstrates how to build a tag cloud.

These samples are open-source and can be downloaded from github

To install and customize the samples on your own server, view the instructions in the readme.

elvis is a digital asset management system. It's main features:

  • Intuitive user interface
  • Easy collaboration and sharing
  • High performance search engine
  • Scalable processing engine to support images, video, office documents, Adobe CS files, PDF, XML and more.
  • Powerful metadata based permission rules

For more information, go to: www.elvisready.com
